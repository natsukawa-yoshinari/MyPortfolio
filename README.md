# Portfolio

WebポートフォリオをPagesで公開するためのリポジトリ

## --- Clicker ---

### 製作時間

2~4時間

### こだわり

購入できるコストにならないとアップグレードのボタンが出現しない。

### 操作方法

Clickボタンを押してScoreを増やす。アップグレードを行う。

### URL
https://natsukawa-yoshinari.github.io/MyPortfolio/Clicker_Game/index.html

## --- Pinball ---

### 製作時間

5~7時間

### こだわり

3つの異なる障害物にあたることスロットが回せられる。スコア増加に数字が昇る演出。

### 操作方法

Aキーで左、Dキーで右のフリッパーが動く。ボタンを押したらすぐ開始。

### URL
https://natsukawa-yoshinari.github.io/MyPortfolio/Pinball_Game/index.html

## --- PapaerSUMO(紙相撲) ---

### 製作時間

3~5時間

### こだわり

力士の重心を手動で調整した点。PostProcessingで被写界深度を設定した点。

### 操作方法

赤側:A, Sキー、青側:k, lキー

### URL
https://natsukawa-yoshinari.github.io/MyPortfolio/PaperSumoGame/index.html
